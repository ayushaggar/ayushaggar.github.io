Project

Kinect V2 Map Visualization

1) Worked on the design of an interface for visualizing maps using the Kinect.
2) Got acquainted with the Kinect and the corresponding additional technologies Web GL, Cesium, C#, Visual Studio.
3) Investigated what kind of gestures can be captured and are meaningful for Map task.
4) Implemented Visual Gesture Builder, bone Joints based gesture and voice recognition.
5) Application allows globe view, street panorama and view from the air through model.
6) User Interface enhanced by speech Interaction and screenshot with various user views on the screen.
